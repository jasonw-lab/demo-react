## Booking Hotel (UI Demo)

Hotel booking experience demo (frontend-only): **search → details → booking → confirmation**.

### Tech stack

- Next.js (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- React Hook Form + Zod
- date-fns, lucide-react

### Setup

```bash
npm install
npm run dev
```

### Structure

- `src/app`: routes and layouts
- `src/components`: shared UI + layout components
- `src/lib`: utilities
- `src/types`: TypeScript types
- `src/data`: mock data

### Notes

This project uses remote placeholder images from `images.unsplash.com` (no API calls).
